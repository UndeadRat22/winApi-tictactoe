# TicTacToe using WinAPI
Made as a practice task for 7th semester univesity course "Programming using Windows API"
## Requirements are as follows:
- Resource scripts: (menus, dialogs, icons).
- UI: labels, buttons, drop down list, etc.
- Graphical elements: show some image based on user input.
- Files: save and restore program status using windows file system.
- DLL: move some code to an external DLL file.
